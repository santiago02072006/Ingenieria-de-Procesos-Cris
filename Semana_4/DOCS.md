Proyecto DevSolve - MVP
Concepto: Ecosistema de software para conectar problemas empresariales con soluciones reutilizables.

Funcionalidades MVP:

Landing Page: Explicación del valor del ecosistema.

Autenticación: Login para 'Cliente' y 'Desarrollador'.

Bounty Board: Formulario para que el Cliente publique problemas.

Catálogo de Soluciones: Marketplace donde se listan soluciones disponibles.

Perfil del Desarrollador: Historial de soluciones creadas y entregadas.

Tecnologías: Next.js (App Router), Tailwind CSS, Supabase.
